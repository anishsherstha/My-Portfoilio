Expense Tracker System

The Daily Expense Tracker is a console-based C application that helps users record and manage their daily expenses. Users can add, view, search, update, and delete expense records. The program also calculates the total expenses and uses file handling to save data permanently so that it is available the next time the program is run.

Tic-Tac-Toe Game

The Tic-Tac-Toe Game is a console-based C application for two players. Players take turns placing X and O on a 3×3 board. The program checks for a winner or a draw, keeps track of player scores, and saves the scores using file handling. It demonstrates fundamental C programming concepts such as arrays, functions, loops, conditional statements, and file handling.
